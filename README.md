# jeykey-ticketing-system

## 🎟️ About the Project

JeyKey Tickets is a dynamic concert seat reservation platform designed to deliver a seamless and interactive ticket booking experience. From exploring live events to selecting preferred seats and securing reservations, this system transforms traditional ticket booking into a modern digital experience.

## ☕ Built with Java

This project was developed using **Java**, applying object-oriented programming principles, event-driven programming, and graphical user interface design to create a functional and user-friendly desktop application.

## 🎓 Academic Milestone

JeyKey Tickets was proudly developed as my **Final Requirement for Computer Programming 2**, showcasing my foundation in Java programming, software design, and real-world application development. This project was recognized for achieving the **highest score in our final requirement**, making it one of the most memorable milestones in my academic and development journey.

## ✨ Key Features

Here are some of the core functionalities of the system:

- 🎫 Browse available concert events
- 💺 Interactive seat selection and reservation
- 🔐 Secure ticket booking process
